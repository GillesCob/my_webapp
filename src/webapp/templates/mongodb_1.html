Description

MongoDB est un système de gestion de base de données NoSQL qui stocke les données sous forme de documents JSON flexibles et hiérarchiques, offrant une évolutivité horizontale, une rapidité de requête et la possibilité de gérer des données non structurées ou semi-structurées.

Les documents sont regroupés dans une Collection qui est elle-même dans une Database.
