Container

Description Code Sources complémentaires

Description

Un container avec Bootstrap est un élément HTML qui enveloppe le contenu de la page, lui appliquant une mise en forme et un espacement appropriés pour assurer une présentation cohérente sur différentes tailles d'écrans.

C'est une notion essentielle afin de réaliser des sites en responsive avec Bootstrap.

Dans la majorité des cas pour le moment on va utiliser container-fluid. 

    <div class="container-fluid">
        <p>Contenu du container</p>
    </div>

Grid system

Description Code Sources complémentaires

Description

Le Grid system de Bootstrap divise l'espace en 12 colonnes, facilitant la création de mises en page flexibles et adaptatives grâce à des classes prédéfinies pour définir la largeur des colonnes et des rangées sur différentes tailles d'écrans.

Nous devons dans un premier temps créer une class "container" puis ensuite une autre "row" afin de mettre en place ce quadrillage.

Quelques exemples :

Box 1 prend 2 col
Box 2 prend 1 col
Box 3 prend 8 col
Place restante

Il est alors possible de modifier la place que prend chaque Box en fonction de la largeur de l'écran (responsive)

Box 1 prend 2 col
Box 2 prend 1 col
Box 4 prend 8 col mais que 3 si on réduit suffisament la fenêtre
Place restante

Dernier exemple avec trois modifications de tailles en fonction de l'écran :

Box 1
Box 2
Box 3
Place restante 
    <div class="row bg-light text-center">
        <div class="col-2 col-lg-4" style="border: 1px solid black;">
            Box 1
        </div>
        <div class="col-1 col-lg-4" style="border: 1px solid black;">
            Box 2
        </div>
        <div class="col-8 col-lg-4" style="border: 1px solid black;">
            Box 3
        </div>
    </div>

Offsets

Description Code Sources complémentaires

Description

L'offset dans Bootstrap permet de décaler le positionnement des éléments dans la grille, offrant une flexibilité pour créer des mises en page asymétriques en ajustant l'espace avant le début d'une colonne spécifique.

Box 1 prend 2 col et est décalée de 3 colonnes vers la droite 
     <div class="col-2 offset-3 bg-light" style="border: 1px solid black;">
        Box 1 prend 2 col et est décalée de 3 colonnes vers la droite
     </div>
